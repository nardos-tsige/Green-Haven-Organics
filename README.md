# Green-Haven-Organics

**live demo**: [live demo](https://green-haven-organics-eight.vercel.app/)
**Figma UI/UX design**: [Figma Design](https://www.figma.com/design/9Bw5UtkKSLNKyfxrHsqT4M/Green-Haven-Organics?node-id=75-886&t=5WRgj8Mxgd5199xr-1)

- Home
